About this product

MyAwesomeLoader is a small - and easy to use - librairie to display personal loaders on your web page.
It has been developed in JavaScript and allow you to create your own Loader based on image or gif.
MyAwesomeLoader is hightly configurable and allow you to choose background color, text color, animations and message you want to display. For any questions or sugestions about this product, contact me to this email adress.

Requirements

MyAwesomeLoader needs following librairies to work perfectly.

  1. Bootstrap (min. V.3.x)
  2. JQuery (min 1.11)

Install

To use this librairie, download the zip archive of version you want and include both JS and CSS file it to you project.

Usage

To define a new loader, you may create new MyAwesomeLoader Oject. Without params, the default launcher is used. To define personalized options, defined them. You might define them all or just afew, leting the others default params. 

                                    /* Définition du loader et de ses options */
                                    var loader = new myAwesomeLoader({
                                        message : 'Loading in progress ...',
                                        imgPath : './img/default.gif',
                                        backgroundColor : '#fff',
                                        textColor: '#000'
                                    });

Options

All the following options might be persolalized to create your own loader.

message Define the loader text String Loading ...
imgPath Define the image/gif path to use String './img/default.gif'
backgroundColor Define the background color String #fff
textColor Define the message color String #000
Care with the img path. Be sure to get the right location.
Height and width are very importent to center image on the screen

Functions

To launch / stop and debug, the following options, use the loader options

                                    var loader = new myAwesomeLoader();
                                    loader.load();
                                    loader.stop();
                                    /*--- [ ... ] --- */
Actions
.load() Active the loader
.() Manually stop the loader
Getters / Setters
getMessage Get current message
getImgPath Get current image path
getBackgroundColor Get current background color
getTextColor Get current text color
setMessage(message) Set new message
setImg(img) Set new image path
setBackground Set new background color
setTextColor Set new text color
